# Images Directory

## Structure

### /gallery/
Place 4 gallery images here with the following names:
- `gallery1.jpg` - Gold reflections
- `gallery2.jpg` - Golden light
- `gallery3.jpg` - Molten gold texture
- `gallery4.jpg` - Jewelry detail

Recommended size: 1920x1080 pixels

### /collection/
Place 4 product images here with the following names:
- `ring.jpg` - Elysian Ring
- `necklace.jpg` - Solstice Necklace
- `bracelet.jpg` - Aura Bracelet
- `earrings.jpg` - Celeste Earrings

Recommended size: 600x800 pixels

## Usage
Images will be automatically loaded from these directories in the Gallery and Collection components.
